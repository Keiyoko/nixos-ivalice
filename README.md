# Nixos-Ivalice
Keio's personal NixOS configuration for his host system: Ivalice, featuring Niri, DankMaterialShell, and Home Manager.

## Install
On a fresh NixOS system, run:

```bash
curl -fsSL https://raw.githubusercontent.com/Keiyoko/nixos-ivalice/main/install.sh | bash
```

After installing, dont forget to change your wallpaper in DMS settings to trigger matugen theming.
